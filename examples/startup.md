# Exemplo: Landing de Startup

## Comando

```bash
/elis full "Landing para NeuralVoice, startup de IA que converte texto em voz ultra-realista" \
  --reference https://elevenlabs.io \
  --stack next \
  --output ./neuralvoice \
  --with-backend
```

## O que o Elis vai fazer

### Phase 1: Discovery (Scout)
- Competidores: ElevenLabs, Play.ht, Murf, Resemble
- Mercado de text-to-speech com IA
- Diferenciais: qualidade, preco, API

### Phase 2: Design (Designer + UXer)
**Visual Direction:**
- Estilo: futurista, tech, premium
- Cores: gradientes escuros, neon accents
- Elementos: waveforms, audio visualizers
- Animacoes: morph effects, particle systems

**Secoes:**
1. Hero com demo interativo
2. Voice samples
3. Features (API, customizacao, etc.)
4. Pricing
5. Use cases
6. CTA final

### Phase 3: Build (Builder + Animator)
**Features especiais:**
- Audio player customizado
- Waveform visualizer com Canvas/WebGL
- Demo de conversao (mockup)
- Animacoes de gradiente

### Phase 4: Backend (Supabase)
- Waitlist: email collection
- API keys management
- Usage tracking
- Auth para dashboard

### Phase 6: Deploy + Launch
- Vercel Edge Functions para API
- Analytics setup
- Product Hunt ready

## Output Esperado

Landing impactante com:
- Visual futurista e premium
- Demo interativo
- Audio players customizados
- Backend para waitlist
- Pronto para lancamento
